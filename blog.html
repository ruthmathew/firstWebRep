Specfic Changes in css3

    Intial Letter

  • Initial letter is a CSS property that selects the first letter of the element and specifies the number of lines occupied by the letter. It is the property where the first letter of the paragraph is written with much higher size than the rest of the content. This property adjusts both the number of lines needed to create the stylized drop cap and the font size. The ::first-letter pseudo-element can be used to select characters that will be formatted as initial letters with a display value of block, table-cell, table-caption, or list-item.

    • Variable Font

  • Variable fonts represent a new set of features where font files are allowed to contain multiple variations of a font in a single file. It allows transitioning font styles, custom font styles, animating and access of the variables contained in the font file. It also gives access to the full range of available styles, weights and widths.

    • Level 2 Grid

  • Level 2 of the grid specification creates the subgrid feature of CSS Grid. One can make grids when using the Grid layout option, a grid element can be transformed into a grid container by setting display: grid on it. The development of browser specifications and implementation is a circular process. So one can watch how this subgrid feature is progressing and of emerging browser implementations.

    • Logical Properties and Values

  • Logical Properties and Values is a CSS module that controls the appearance through logical directions and dimension mappings. It uses the terms like block and inline to describe the direction in which they flow. The Logical Properties and Values specification characterizes mappings for physical values in their logical relations.
